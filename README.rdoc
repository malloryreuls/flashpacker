== Flashpacker

===Helping backpackers travel the world like a local

<b>Created with</b> Ruby on Rails, PostgreSQL, Foundation, and Google Maps Javascript API

A location-based app that allows backpackers to find, recommend and upvote fun local spots on a map as they travel.

<b>Created by</b> {Mallory Reulman}[http://www.github.com/malloryreuls] and {Kelsey Ledford}[http://www.github.com/kelseyledford]
